API Examples:
Chat endpoint:
curl -X POST "https://syftbox.net/api/v1/send/msg?x-syft-url=syft%3A%2F%2Faggregator%40openmined.org%2Fapp_data%2Fclaude-sonnet-3.5%2Frpc%2Fchat&x-syft-from=guest@syft.org" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "tell me about dogs"}
    ],
    "model": "tinyllama:latest",
    "user_email": "aggregator@openmined.org"
  }'
Search endpoint:
curl -X POST "https://syftbox.net/api/v1/send/msg?x-syft-url=syft%3A%2F%2Faggregator%40openmined.org%2Fapp_data%2Fsearch%2Frpc%2Fsearch&x-syft-from=guest@syft.org" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "your search query",
    "top_k": 5,
    "user_email": "aggregator@openmined.org"
  }'
