Syft Protocol

The SyftBox Protocol is a pluralistic protocol that turns isolated local folders, called datasites, into nodes in a secure, distributed file and compute network. Think Dropbox, but extended across organizations, with strong guarantees around privacy, control, and governance.

This architecture enables a new mode of collaboration:

Global-Scale AI

Data stays local but can still participate in global-scale AI

Programmable Governance

Governance is programmable, enforced by protocol, not trust

Explicit Access

No silent sharing, no backdoors‚Äîaccess is explicit

Attribution by Design

Computation is attributable, auditable, and runs at the source

Each datasite is fully owned and operated by its host. Data never moves or gets copied unless explicitly allowed. Instead, others can propose computations, such as AI model inferences or data processing jobs, that run where the data lives.

All proposals are subject to granular, network-wide permissioning and approval flows. The network is operated through an e2e encrypted server gateway that facilitates communication between nodes, fully secure.

Syft Routers

As easy as deploying FastAPI, backed by built-in governance, privacy, and full audit trails.

Syft Routers are network gateways that let you safely offer AI endpoints, such as search or chat, on top of your private data and models, without revealing the underlying assets.

From a retrieval model on your laptop to sensitive medical data in an air-gapped environment, routers facilitate standardized, permissioned access to any resource behind a SyftBox datasite, enforcing schema consistency and privacy by default.

Key Features

Unified Schema (OpenAI-Compatible)

All routers expose endpoints that follow a shared spec (chat, search), making them easy to consume from any client or LLM‚Äîno custom adapters required.

Any Backend: Local or Remote

Routers don't care if the data/model lives in-memory, on disk, behind a firewall, or across the world. They proxy requests and responses, while enforcing policy.

Governance Built In

Offline-Tolerant by Design

Routers run on syftRPC, a resilient, asynchronous RPC protocol built to survive flaky networks and reconnect automatically.



Two Interoperable Paradigms

Syft Routers let data and model owners expose standardized, permissioned AI endpoints, search and chat, across private resources. These enable two core paradigms in decentralized AI systems, both with built-in attribution and full owner control.
